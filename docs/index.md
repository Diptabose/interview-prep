# Interview Prep

Welcome to my personal interview-prep knowledge base. Use the **search** (top bar) to jump straight to a concept, or browse by domain in the left sidebar.

## Domains

Notes are organized by domain, then nested as deep as the topic warrants:

- **js** — JavaScript language and runtime
- **frontend** — React, hooks, rendering, browser
- **backend** — Node, NestJS, APIs, auth
- **databases** — SQL, MongoDB, indexing
- **system-design** — scalability, caching, queues
- **cicd** — GitHub Actions, pipelines, deployment
- **dsa** — data structures & algorithms

> New notes are written with the `/prep` command and appear here automatically after each push.
